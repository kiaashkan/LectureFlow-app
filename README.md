# LectureFlow Android 📱

Official Android application for LectureFlow.

Access AI-powered lecture analysis directly from your Android device.

Upload lecture recordings and automatically receive:

* Transcript
* Technical terms
* Formula extraction
* AI-generated summaries

Built for students, researchers, educators, and anyone who wants AI-assisted learning on mobile.

## Features

* 🎙 Upload lecture recordings from your Android device
* 📄 View AI-generated transcripts
* 💊 Browse extracted technical terms
* 🧪 Review detected formulas
* 📋 Read AI-generated summaries
* ⚡ Fast and responsive interface
* 🌙 Dark / Light mode support
* 📱 Mobile-first experience
* 🔓 Fully open source

---

## Development

LectureFlow Android was developed using Android Studio and Java, with assistance from Claude AI during development.

The application is designed to provide seamless access to the LectureFlow platform and make lecture analysis available directly on Android devices.

### Technologies

* Java
* Android Studio
* Material Design
* LectureFlow API
* Claude AI (development assistance)
