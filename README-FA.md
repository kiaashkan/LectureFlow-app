# LectureFlow Android 📱

اپلیکیشن رسمی اندروید LectureFlow

با استفاده از این اپ می‌توانید مستقیماً از طریق گوشی اندرویدی خود به امکانات تحلیل هوشمند سخنرانی و فایل‌های صوتی دسترسی داشته باشید.

فایل صوتی یا ضبط کلاس را آپلود کنید و به‌صورت خودکار دریافت کنید:

* متن کامل گفتار (Transcript)
* استخراج اصطلاحات تخصصی
* استخراج فرمول‌ها
* خلاصه‌سازی هوشمند با هوش مصنوعی

این برنامه برای دانشجویان، پژوهشگران، اساتید و تمامی افرادی که به یادگیری و تحلیل محتوای صوتی نیاز دارند طراحی شده است.

## امکانات

* 🎙 آپلود فایل‌های صوتی و ضبط کلاس از گوشی
* 📄 مشاهده متن تولیدشده توسط هوش مصنوعی
* 💊 استخراج و نمایش اصطلاحات تخصصی
* 🧪 تشخیص و نمایش فرمول‌های موجود در محتوای آموزشی
* 📋 تولید خلاصه هوشمند از جلسات و سخنرانی‌ها
* ⚡ رابط کاربری سریع و روان
* 🌙 پشتیبانی از حالت روشن و تاریک
* 📱 طراحی‌شده برای موبایل
* 🔓 کاملاً متن‌باز (Open Source)

## توسعه

LectureFlow Android با استفاده از Android Studio و زبان Java توسعه داده شده است و در فرایند توسعه از Claude AI نیز کمک گرفته شده است.

هدف این برنامه فراهم کردن دسترسی سریع و آسان به امکانات LectureFlow روی دستگاه‌های اندرویدی است.

### فناوری‌های استفاده‌شده

* Java
* Android Studio
* Material Design
* LectureFlow API
* Claude AI
